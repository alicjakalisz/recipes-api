## Recipes API
This repository contains an example of a recipes API. The operations that it allows are:

### GET /recipes  

### GET /recipes/${id}  

### POST /recipes
'''
{
   "name"
   "origin"
   "description"
   "ingredients"  [ {
        name
        amount
    }]
   "tags" []
   "steps"
}
'''

### UPDATE /recipes

### DELETE /recipes/${id}
